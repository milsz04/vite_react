React Article Dashboard
A modern, responsive web application built to practice advanced React concepts. This project implements a Master-Detail interface for managing a collection of articles and categories.

Key Features:

Tech Stack: Built with React and Vite for fast performance.

Routing: Uses react-router-dom to manage navigation between lists, details, and forms without page reloads.

CRUD Operations: Create, Read, Update, and Delete articles and categories.

Data Persistence: Simulates an asynchronous backend API using LocalStorage, so your data survives page reloads.

Layout: Responsive CSS Grid layout with a clean, modern UI.
